PowerBI Terms
====================
DataSource  :  data can be loaded from following sources
			-file system 
				-text file
				-csv (flat file)
				-excel
				-ms access
				-json
				-data
				-sdata
				etc.
			-Server
				-MS SQL Server
				-Oracle
				-IBM DB
				-Google Sheet(Drive)
				-SAP
				-Hadoop
				-SSAS
				etc.

					 
Extension:
		pbix			

				
Table	: combination of row and table
Column   : field 

Report  : Visualiztion 
Data    : details row
Relationships:  how two and more than two tables connected with each other 

Number/Numeric Column : is also known as measure
Text Column     : is also known as dimenssion 


Reports:
	Cards			: we can show the summary 
	Multiple Cards		: we can show summary by category
	Pie			: will not allow negative value 
	Bar/Column Chart 
			-> Vertical
					->Single Bar
					->Stacked
					->Group  (side)
			-> Horizental 
					->Single Bar
					->Stacked
					->Group  (side)
